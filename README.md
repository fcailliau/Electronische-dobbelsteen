# Electronische-dobbelsteen
Project opkomende technologie; ardruino project; maken electronishe dobbelsteen

Stappenplan gebruiker:

1 Druk op de knop

2 Een aantal leds (1-6) licht op, het aantal leds dat oplicht is gelijk aan het gerolde nummer.

Deze dobbelsteen is een d6 (een zeszijdige dobbelsteen), de standaard dobbelsteen bij de meeste gezelschapspelltjes.
